🎨 Equity Lens — Design Refresh

Audit of current UI + proposed improvements. I'll show each section one at a time.

Current Pain Points

Issues

  • No design system — every component styles itself
  • Hand-rolled SVG chart — no interactivity, no zoom, no tooltips
  • Flat table layouts — no visual hierarchy on the dashboard
  • No dark mode — financial apps are expected to support it
  • No navigation bar — hard to move between sections
  • Raw colors (green/red) without a cohesive palette
  • No loading states, no empty states, no error states

What Works

  • Functional layout covers all features
  • Factor score bars with color coding
  • Bull/Base/Bear scenario cards
  • Signal history table is clean
  • Alert severity colors are intuitive

Proposed Stack

1

Tailwind CSS v4

Utility-first CSS framework. Ships with Next.js natively. Dark mode via dark: prefix. Consistent spacing, typography, and color system.

2

TradingView Lightweight Charts

Industry-standard financial charting library. Interactive crosshair, volume bars, time range selector, MA overlays. Free and lightweight (42KB gzip).

Design Direction

Professional financial terminal aesthetic — dark background (#0d1117), clean cards with subtle borders, monospace for data, sans-serif for UI text. Inspired by Bloomberg Terminal and TradingView but simplified for retail investors.
