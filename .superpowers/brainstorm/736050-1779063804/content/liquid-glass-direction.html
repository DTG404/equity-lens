🧊 Apple Liquid Glass — Design Direction

Translating Apple's design language to a financial terminal

Core Principles (from Apple HIG)

🔍
Clarity
Text must be legible over glass. Use high contrast, bold weight for data, never place critical text over busy backgrounds.
👁️
Deference
Content is king. Glass UI recedes — it's a functional layer that floats above data without competing for attention.
📐
Depth
Visual layers create hierarchy. Glass sits between the background (content) and the user, with controls on top.

Apple's Layering System

Layer 3 Interactive controls — buttons, sliders, inputs, toggles
Layer 2 Liquid Glass — navigation, sidebars, tab bars, cards, panels
Layer 1 Content — charts, tables, data, background

Translating to Web — Tech Options

RECOMMENDED
Glin UI
77 React components built on Radix + Tailwind. Glass, liquid, matte, glow surface variants. 5 elevation levels. MIT license. Same ergonomics as shadcn/ui. Ships tokens, motion, and dark mode.
npm: @glinui/ui @glinui/tokens
Custom CSS (Tailwind)
Use Tailwind's backdrop-blur utilities with custom color tokens. Full control, smaller bundle, no dependency. More work to build components from scratch but no vendor risk.
backdrop-blur-xl bg-white/5 border-white/10

Implementation Approach

My recommendation: Glin UI for rapid component coverage + custom CSS for the unique bits

✅ Recommended: Glin UI
• 77 components with glass/liquid/matte/glow variants
• Built on Radix UI (accessible by default)
• Tailwind CSS v4 — fits our stack perfectly
• 5 elevation levels for depth hierarchy
• MIT licensed, open source
• Dark mode built in
• ~same ergonomics as shadcn/ui (CLI install)
Alternative: Custom CSS
• Zero dependencies
• Full control over every pixel
• Use Tailwind's backdrop-blur utilities
• More work — every component hand-built
• No pre-built Radix accessibility

Phase plan: 1) Install Glin UI + tokens → 2) Set up theme (colors, glass tokens, fonts) → 3) Build nav shell with floating glass nav bar → 4) Restyle dashboard components → 5) Install TradingView Lightweight Charts → 6) Restyle research page → 7) Polish
